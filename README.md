# Shakespeare Gastropub

Modern, responsive website for Shakespeare Gastropub in Limassol, Cyprus. Features a menu image carousel, downloadable menus, Google Maps integration, and a warm heritage design.

## Features

- Menu showcase with image carousel (main menu + breakfast menu)
- Downloadable PDF menus
- Google Maps integration showing pub location
- Social media links (Facebook)
- Contact section with phone, email, and address
- Responsive design with sticky header on scroll

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** (dev server and build)
- **Express** (production server)
- **Tailwind CSS v4**
- **Framer Motion** for animations
- **Radix UI** primitives (shadcn/ui)
- **Google Maps** integration
- **Wouter** for client-side routing

## Setup

```bash
pnpm install
pnpm dev       # Start dev server
pnpm build     # Production build
pnpm start     # Start production server
pnpm check     # Type check
```

## Design

- **Style:** Warm Heritage Modern
- **Colors:** Deep Burgundy (`#8B3A3A`), Warm Cream (`#F5F1E8`), Soft Gold (`#D4AF37`)
- **Typography:** Playfair Display (headings), Lato (body), Cormorant (accents)
